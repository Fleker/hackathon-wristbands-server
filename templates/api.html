ProfHacks 2016 Recommended APIs

There numerous different APIs that given you many different possibilities. We have create a small list of APIs (including our own) that either links or describes each one.

Some that we recomend are:

  • Twillio
    • We have API keys at the Info Desk
  • Kloudless
    • We have API keys at the Info Desk
  • Clarifai
    • We have API keys at the Info Desk
  • Live ProfHacks Data
    • Using wristbands we gave out at registration we are creating a live data set during ProfHack
    • All data is anonymous
    • View Registration Data
      • GET Method
      • When registering for ProfHacks, each individual is required to sign in using their given NFC tag. When doing this, our application records their serial number as well as their time of registration.
      • JSON Format
        [
             {
                 "id":Integer,
                 "serial":"NFC Tag Serial Number",
                 "timestamp":"Time of Registration"
             },{...},...
        ]
    • View Event Data
      • GET Method
      • When participating in ProfHacks events, each individual will scan in using their given NFC tag. When doing this, our application records their serial number, time of check-in, and the event they are attending.

        The events we will be recording are:
        • Opening Ceremony
        • Team-building Session
        • Journalism Session
        • CloudMine Session
        • Dinner
        • Midnight Snack
        • Game Tournament
        • Late-Night Snack
        • Breakfast
        • Lunch
      • JSON Format
        [
             {
                 "id":Integer,
                 "serial":"NFC Tag Serial Number",
                 "timestamp":"Time of check in to event",
                 "message":"Attended Event"
             },{...},...
        ]
    • All timestamps are 5 hours ahead of EST
  • Filestack
  • Plotly
  • Bitalino
  • Myo
