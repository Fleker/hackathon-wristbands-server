ProfHacks 2016 Recommended APIs

There numerous different APIs that given you many different possibilities. We have create a small list of APIs (including our own) that either links or describes each one.

Some that we recomend are:

  • Twillio
    • We have free credits at the Info Desk
  • Kloudless
    • Register here for credits
    • Kloudless is a universal API that allows you to build with many file storage services like Dropbox, Google Drive, and Evernote by coding just one time.
  • Estimote Beacons
    • To change info in the Estimote Cloud, you'll need to log in.
    • Use `felkern0@students.rowan.edu` with the password `ProfH@cks`
  • Get a .tech domain
    • Go to www.get.tech/startups
    • Select ProfHacks from the list when requesting the name
    • For queries, you may contact mitsu@radix.email or Suman Das
  • NameCheap
    • Get a domain name through NameCheap. Check out the cards that we're handing out.
  • FileStack
    • Connect to all your cloud storage services
    • Register for a plan
    • Double the limits of the free plan
      • 500 uploads
      • 1000 transformations
      • 3GB of bandwidth

  • Live ProfHacks Data
    • Using wristbands we gave out at registration we are creating a live data set during ProfHack
    • All data is anonymous
    • View Registration Data
      • GET Method
      • When registering for ProfHacks, each individual is required to sign in using their given NFC tag. When doing this, our application records their serial number as well as their time of registration.
      • JSON Format
        [
             {
                 "id":Integer,
                 "serial":"NFC Tag Serial Number",
                 "timestamp":"Time of Registration"
             },{...},...
        ]
    • View Event Data
      • GET Method
      • When participating in ProfHacks events, each individual will scan in using their given NFC tag. When doing this, our application records their serial number, time of check-in, and the event they are attending.

        The events we will be recording are:
        • Opening Ceremony
        • Team-building Session
        • Journalism Session
        • CloudMine Session Slideshow Karaoke
          • Still refered to as CloudMine Session in API
        • Dinner
        • Midnight Snack
        • Game Tournament
        • Late-Night Snack
        • Breakfast
        • Lunch
      • JSON Format
        [
             {
                 "id":Integer,
                 "serial":"NFC Tag Serial Number",
                 "timestamp":"Time of check in to event",
                 "message":"Attended Event"
             },{...},...
        ]
    • All timestamps are 5 hours ahead of EST and 4 hours ahead of EDT
